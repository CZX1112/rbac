# rbac

#### 介绍
一个简易的rbac管理工具，目前正在开发中，已支持一键部署

#### 软件架构
软件架构说明


#### 安装教程

部署指令如下：
在某个空文件夹下
git clone https://gitee.com/he-lingzhou/rbac.git

cd rbac

chmod +x deploy.sh

./deploy.sh

即可运行，注意，根据申请的服务器ip，需要修改my-admin-project目录下的nginx.conf内容，将server_name后的ip改成服务器的ip
